## 前端状态机
```text
LOCKED
  -> 激活成功 -> HUMAN_MODE

HUMAN_MODE
  -> 点击“智能体控制台” -> AGENT_MODE
  -> 继续输入中文/ID+数字串 -> HUMAN_MODE

AGENT_MODE
  -> 载入握手模板 -> AGENT_MODE
  -> 预演握手成功 -> AGENT_READY
  -> 返回人类通道 -> HUMAN_MODE

AGENT_READY
  -> Token 失效 -> PAYMENT_REQUIRED
  -> 返回人类通道 -> HUMAN_MODE

PAYMENT_REQUIRED
  -> 完成 X402 支付 -> AGENT_READY
  -> 切回人类通道 -> HUMAN_MODE
```

## UI原则
- 人类通道保持当前黑盒体验，不破坏中文转数字主流程
- 机器通道以 JSON 窗口承载，不和人类输入框混用
- 先本地预演，再接真实后端
