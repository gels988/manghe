# [双轨制聊天室] 网关层双轨路由与状态机

## 网关伪代码
```text
function gateway(request):
    token = parseAuthToken(request)
    paymentReceipt = parseX402Receipt(request)
    humanVoucher = parseHumanVoucher(request)

    if token.kind == "human" and verifyHumanToken(token):
        return routeToHumanChat(request, token)

    if token.kind == "agent" and verifyAgentToken(token):
        return routeToAgentHandshake(request, token)

    if humanVoucher and verifyHumanVoucher(humanVoucher):
        humanToken = issueHumanToken(humanVoucher, request.identity)
        return {
            status: 200,
            route: "human-chat",
            issue_token: humanToken
        }

    if paymentReceipt and verifyX402Receipt(paymentReceipt):
        agentToken = issueAgentToken(paymentReceipt, request.agent.id)
        return {
            status: 200,
            route: "agent-handshake",
            issue_token: agentToken
        }

    return {
        status: 402,
        route: "x402-paywall",
        payment_required: {
            scheme: "x402",
            quote_id: createQuote(request),
            accept: ["x402"]
        }
    }
```

## 双轨状态机
```text
[INIT]
  -> 检查 Human_Token
     -> 有效 -> [HUMAN_READY] -> 人类聊天模块
  -> 检查 Agent_Token
     -> 有效 -> [AGENT_READY] -> 智能体握手模块
  -> 检查人类捐赠凭证
     -> 有效 -> 签发 Human_Token -> [HUMAN_READY]
  -> 检查 X402 支付回执
     -> 有效 -> 签发 Agent_Token -> [AGENT_READY]
  -> 否则 -> [PAYMENT_REQUIRED_402]
```

## 状态含义
- INIT: 请求进入网关，尚未分流
- HUMAN_READY: 已完成人类凭证识别，进入万维码加密聊天
- AGENT_READY: 已完成 X402 支付握手，进入机器 JSON 通道
- PAYMENT_REQUIRED_402: 未持有可用凭证，返回 402 支付要求
