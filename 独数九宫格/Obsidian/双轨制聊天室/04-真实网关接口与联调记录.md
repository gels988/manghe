# [双轨制聊天室] 真实网关接口与联调记录

## 服务
- 静态页: `http://<host>:8000`
- 信令: `ws://<host>:8787`
- 真实网关: `http://<host>:8790`

## 接口

### `GET /health`
- 作用: 返回网关健康状态
- 返回:

```json
{
  "status": "ok",
  "service": "aim2m-real-gateway",
  "port": 8790,
  "quotes": 0
}
```

### `POST /paywall`
- 作用: 为智能体生成真实 `402` 支付要求
- 请求:

```json
{
  "agent": {
    "id": "X402-DEMO-001"
  }
}
```

- 返回: `HTTP 402`

```json
{
  "version": "x402-handshake-v1",
  "status": "payment-required",
  "route": "x402-paywall",
  "paywall": {
    "http_status": 402,
    "required_scheme": "x402",
    "quote_id": "quote_xxx",
    "amount": "0.001",
    "asset": "USDT",
    "network": "BASE",
    "expires_at": "2026-06-10T03:10:10.673Z",
    "pay_to": "0x86524871F310c8Cf70b824c9e96Fe235f16CbF38",
    "memo": "AIM2M_AGENT_X402-DEMO-001"
  },
  "dev_receipt_example": {
    "scheme": "x402",
    "quote_id": "quote_xxx",
    "payer": "local-agent",
    "proof": "sha256-proof"
  }
}
```

### `POST /handshake`
- 作用: 统一处理人类凭证、人类离线激活码、X402 回执与 Agent_Token

#### 人类请求

```json
{
  "version": "x402-handshake-v1",
  "route": "human",
  "human": {
    "identity": "18080860669",
    "identity_type": "local-profile",
    "voucher_code": "AIM2M-4538-FA4E-OFFLINE"
  },
  "auth": {
    "presented_token": null
  },
  "payment": {
    "mode": "none",
    "receipt": null,
    "quote_id": null
  }
}
```

#### 机器请求

```json
{
  "version": "x402-handshake-v1",
  "route": "agent",
  "agent": {
    "id": "X402-DEMO-001",
    "role": "crawler",
    "intent": "handshake",
    "capabilities": ["json-exchange"]
  },
  "auth": {
    "presented_token": null
  },
  "payment": {
    "mode": "x402",
    "receipt": null,
    "quote_id": null
  },
  "session": {
    "nonce": "n1",
    "reply_format": "json"
  }
}
```

## 联调记录
- `GET /health` 已返回 `200`
- `POST /handshake` 无回执时已返回真实 `402`
- 把 `dev_receipt_example` 回填到 `payment.receipt` 后，已返回 `HTTP 200` 与真实 `Agent_Token`
- 人类通道提交 `identity + AIM2M-4538-FA4E-OFFLINE` 后，已返回 `HTTP 200` 与真实 `Human_Token`

## 当前实现边界
- 现在的 X402 验证为开发期真实网关 + 本地测试回执校验，不是链上最终清算
- Token 为 HMAC 签发，可用于本地与局域网阶段联调
- 下一阶段可继续接入真实支付网关、账本落库与 token 撤销表
